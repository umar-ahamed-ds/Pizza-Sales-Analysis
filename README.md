# Pizza Sales Analysis Dashboard

## Project Overview

This project analyzes pizza sales data using SQL and Power BI to uncover business insights, sales trends, customer ordering patterns, and best/worst-selling pizzas.

## Tools Used

- SQL
- Power BI
- Excel
- DAX
- Power Query

## Key KPIs

- Total Revenue
- Average Order Value
- Total Orders
- Total Pizzas Sold
- Average Pizzas Per Order

## Dashboard Features

### Sales Performance Dashboard
- Daily Sales Trend
- Monthly Sales Trend
- Sales by Category
- Sales by Size
- Total Pizzas Sold by Category

### Best/Worst Sellers Dashboard
- Top 5 Pizzas by Revenue
- Top 5 Pizzas by Quantity
- Top 5 Pizzas by Orders
- Bottom 5 Pizzas by Revenue
- Bottom 5 Pizzas by Quantity
- Bottom 5 Pizzas by Orders

## Business Insights

- Classic Pizza category generated the highest sales.
- Large-sized pizzas contributed the highest revenue.
- Friday and Saturday evenings showed peak order volume.
- July and January recorded the highest sales activity.
